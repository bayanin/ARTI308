## Dataset Summary

**Dataset:** Healthcare Stroke Prediction Dataset  
**Format:** CSV (tabular data)

This dataset contains medical and lifestyle information about individuals, including features such as age, gender, hypertension, heart disease, average glucose level, BMI, and smoking status.

---

## Machine Learning Problem

**Problem Type:** Classification (Binary)

The goal of this project is to predict whether a person is likely to have a stroke based on the available medical and lifestyle features.

---

## Target Variable

**Target:** `stroke`  
- `0` → No stroke  
- `1` → Stroke

---

## Dataset Source

The dataset was obtained from Kaggle.
